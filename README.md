# 🤖 Null-Hunters Bot Control Panel

Panel gráfico en Python para controlar un bot de Discord de forma sencilla usando una interfaz visual.

---

## 📌 Características

* Interfaz gráfica con Tkinter
* Inicio de bot mediante token
* Creación automática de canales
* Creación de roles personalizados
* Envío masivo de mensajes
* Fondo personalizado con imagen

## 🚀 Uso

1. Clona este repositorio:

```bash
git clone https://github.com/tuusuario/null-hunters-bot-panel.git
```

2. Entra en la carpeta:

```bash
cd null-hunters-bot-panel
```

3. Ejecuta el programa:

```bash
python tuarchivo.py
```

4. Introduce:

* Bot Token
* Bot ID

5. Haz clic en **Iniciar Bot**
